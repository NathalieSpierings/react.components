# BasicLayout Component

De `BasicLayout` component biedt een eenvoudige pagina-layout zonder sidebars.  
Het combineert een header, content-area en optionele pagina loader.

## Belangrijkste functionaliteiten

- **Pagina loader:** toont een `PageLoader` component wanneer `loading` true is.
- **Header ondersteuning:**  
  - Toont `pageTitle` en breadcrumbs via `Header` component.  
  - Header kan worden verborgen via layout context (`showHeader`).
- **Content area:**  
  - `Outlet` wordt gebruikt om nested routes te renderen.
- **Layout context integratie:**  
  - Respecteert `fullscreen` en `showHeader` vanuit `LayoutContext`.
- **SVG sprite integratie:**  
  - Inclusief globale `SvgSprite` voor iconen.
