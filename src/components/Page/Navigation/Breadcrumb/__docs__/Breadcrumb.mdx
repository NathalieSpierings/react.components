# Breadcrumb Component

De `Breadcrumb` component toont een hiërarchische navigatie, zodat gebruikers kunnen zien waar ze zich bevinden in de applicatie en eenvoudig kunnen terugkeren naar vorige pagina’s.

---

## Belangrijkste kenmerken

- **Hiërarchische navigatie**: toont een pad van links naar de huidige pagina.
- **Actieve pagina markering**: de laatste breadcrumb wordt gemarkeerd als actieve pagina.
- **Toegankelijk**: ondersteunt `aria-label` voor screenreaders.
- **Flexibel gebruik**: items worden dynamisch gegenereerd via een array van objecten met `label` en `href`.
