# Blockquote Component

Het **`Blockquote`** component wordt gebruikt om een citaat of opvallende tekst te markeren, vaak met een optionele bronvermelding.

## Belangrijkste functionaliteiten

- **Citaatweergave**: Weergeeft een bloktekst (`text`) als citaat.  
- **Bronvermelding**: Optionele `citeText` om de auteur of bron van het citaat te tonen.  
- **Eenvoudig gebruik**: Minimalistische structuur voor snelle integratie in content.
