# DividerSplitted Component

`DividerSplitted` is een component dat een visuele scheidingslijn met een label rendert. Het label wordt in het midden geplaatst en de lijn kan optioneel van kleur worden voorzien.

## Belangrijkste functionaliteiten

- **Label in het midden:** Het opgegeven label wordt gecentreerd tussen de scheidingslijnen.  
- **Kleuraanpassing:** Tekst- en randkleur kunnen worden aangepast met `ColorDefinitions`.  
- **Custom CSS:** Extra styling kan meegegeven worden via `dividerSplittedCss`.
